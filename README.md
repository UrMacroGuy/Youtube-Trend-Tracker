🧐 Why This Exists?
Most creators fail because they upload into a vacuum. While everyone else is busy checking their own Studio dashboard, you’re busy dissecting the entire competition. YT-NichePulse is a strategic command center for creators who want to stop "guessing" and start dominating.

⚡ Key Features (The "Alpha" Specs)
📈 1. Competitor Stealth-Tracking
Engagement Velocity: Don't just watch sub counts. Track how fast a video is moving in its first 24 hours.

Outlier Alerts: Get notified the second a competitor's video performs 300% better than their average.

🕵️ 2. The Content Gap Radar
High Demand / Low Supply: Our algorithm scans the niche to find topics people are searching for but no one is making good videos on.

Keyword Saturation: See exactly which tags are "overcooked" and which ones are wide open for ranking.

🖼️ 3. Visual Intelligence
Thumbnail Pattern Match: Scrape the top 100 thumbnails in your niche to see which colors and fonts are currently "clicking."

Retention Loops: Track which background aesthetics (Lofi, Phonk, or high-speed gameplay) are keeping viewers glued to the screen.

🛠️ The Tech Stack
Built for speed, accuracy, and deep-dive analytics.

Engine: Python 3.11+ / Node.js

Data Source: YouTube Data API v3

Database: SQLite (Lightweight & Portable)

UI: Streamlit / React (Clean, Dark-Mode ready)
