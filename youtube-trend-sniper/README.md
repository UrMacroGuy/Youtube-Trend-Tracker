# YouTube Trend Sniper 🎯

Find the best faceless, voiceless niches to grow your YouTube channel!

## 🚀 How to Run

### Option 1: Double-click (Easiest)
Just double-click `run.bat` and the analysis will start!

### Option 2: Command Line
```
python app.py
```

## 📊 What It Does

- Analyzes 10+ faceless-friendly niches
- Tracks view growth rates (48-hour focus)
- Identifies hot opportunities for AI-generated content
- Saves results to JSON files for tracking trends
- Provides content ideas for top niches

## 🎯 Faceless Niches Tracked

1. **Gaming** - Gameplay, walkthroughs, no commentary
2. **Tech** - Tutorials, reviews, screen recordings
3. **Finance** - Charts, crypto, stock analysis
4. **AI Art** - Midjourney, Stable Diffusion showcases
5. **Educational** - Facts, explainers, animations
6. **ASMR** - Relaxing sounds, ambient content
7. **Motivation** - AI voice quotes, success stories
8. **Compilations** - Best moments, funny clips
9. **Music** - Lofi beats, study music
10. **Animation** - Short stories, motion graphics

## 📈 Understanding Results

- **🔥 HOT** - Top 3 niches with explosive growth
- **⬆️ RISING** - Strong momentum, good opportunity
- **📊 STABLE** - Consistent performance

**Trend Score** = Views ÷ Days since published (higher = better)

## 💡 Pro Tips

1. **Run daily** to track trend changes
2. **Focus on HOT niches** for fastest growth
3. **Use AI tools** like:
   - Video generation: Runway, Pika, Kaiber
   - Voice: ElevenLabs, Murf.ai
   - Scripts: ChatGPT, Claude

4. **Content types that work:**
   - Lists (Top 10, Best X)
   - Comparisons (A vs B)
   - How-to tutorials
   - News and updates
   - Compilation videos

## 🔧 Technical

- Built with pure Python
- Uses YouTube Data API v3
- No external dependencies (except requests)
- Saves analysis history
